
### Простой чат-бот на WPF с поддержкой базовых команд и сохранением истории.

---

## Возможности

- Приветствия и прощания
- Вопросы о времени и дате
- Арифметические операции (сложение, вычитание, умножение, деление)
- Статистика общения
- Сохранение истории чата в JSON

---

## Примеры команд

| Команда | Пример | Ответ |
|---------|--------|-------|
| Приветствие | `привет` | Привет, Имя! |
| Прощание | `пока` | Пока, Имя. До встречи! |
| Время | `который час?` | Сейчас 14:30:00 |
| Дата | `какое сегодня число` | Сегодня 25 января 2026 года |
| Сложение | `сложи 10 и 5` | Результат: 15 |
| Вычитание | `вычти 20 из 8` | Результат: 12 |
| Умножение | `умножь 7 на 3` | Результат: 21 |
| Деление | `раздели 100 на 4` | Результат: 25 |
| Статистика | `статистика` | Всего сообщений: 10... |

---

## Структура проекта

```
chat-bot-on-C
/
├── App.xaml              # Настройки приложения
├── App.xaml.cs           # Точка входа
├── LoginWindow.xaml      # Окно авторизации (разметка)
├── LoginWindow.xaml.cs   # Окно авторизации (логика)
├── MainWindow.xaml       # Главное окно чата (разметка)
├── MainWindow.xaml.cs    # Главное окно чата (логика)
├── ChatBot.cs            # Класс чат-бота
├── Message.cs            # Класс сообщения
└── chat_history.json     # История сообщений (создаётся автоматически)
```
